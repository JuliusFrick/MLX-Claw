## Prompt Structure Template

```
[ROLE]: You are a [SPECIFIC EXPERT]
[CONTEXT]: The user will provide [INPUT TYPE]
[TASK]: Help them [DESIRED OUTPUT]
[CONSTRAINTS]:
- [RULE 1]
- [RULE 2]
[OUTPUT FORMAT]:
[EXAMPLE]:
```

## Quality Checklist

- [ ] Clear role definition
- [ ] Specific output format
- [ ] 2-3 examples included
- [ ] Edge cases covered
- [ ] Tips for best results
- [ ] Variation options

3. "Create [format] for [platform]" prompts
4. "Transform [input] into [output]" prompts

4. Best practices
5. Related prompts (upsell)
